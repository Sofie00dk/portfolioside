Server

Skal man sætte en website op på internettet skal man bruge en server. En server deler sine ressourcer og filer med andre computer
En server står i et serverrum, dog behøver man ikke at have en server, man kan installere programmer der gør det samme. Når vi prøver at finde ting på nettet, gør vi det via IP.
En webserver kan indeholde flere server, Serveren har en IP-adresse (eks. 102.10.5.245) Webserveren leverer HTML, CSS og Javascript samt billeder mv. Til klienter (browser) En browser kan anvende IP-adresse eller domæne.
